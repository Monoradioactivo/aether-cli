# @aetherpush/cli

Command-line interface for [Aether](https://aetherpush.com) — over-the-air updates for React Native apps.

> **Pre-release.** This package is under active development. The first stable release is planned for Q1 2026.

## Install

```sh
npm install -g @aetherpush/cli
```

Requires Node.js 18 or later.
